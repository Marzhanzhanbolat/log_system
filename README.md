# 📌 Log System (Лог жүйесі)

## 🎯 Мақсаты

Бұл жобаның мақсаты — бағдарламаның жұмыс барысында орын алатын барлық маңызды әрекеттерді файлға жазу (лог жүргізу жүйесін құру).

Лог жүйесі арқылы:

* бағдарламаның қалай жұмыс істеп тұрғанын бақылауға болады
* қателерді оңай табуға болады
* қолданушы әрекеттері тіркеледі

---

## 📖 Жобаның сипаттамасы

Бұл жоба Java тілінде жазылған қарапайым лог жүйесі болып табылады.
Бағдарлама әрбір орындалған әрекетті арнайы файлға (`log.txt`) сақтайды.

Әрбір жазбада:

* әрекет түрі (INFO, LOGIN, UPDATE, ERROR)
* хабарлама
* уақыт белгісі болады

---

## ⚙️ Жұмыс істеу принципі

1. Бағдарлама іске қосылады
2. `log()` функциясы шақырылады
3. Ағымдағы уақыт алынады (`LocalDateTime`)
4. Мәліметтер файлға жазылады (`FileWriter`)
5. Барлық логтар `log.txt` ішінде сақталады

## ❓ Сұрақтар және жауаптар

### 🔹 Лог деген не?

Лог — бұл бағдарламада орын алған барлық оқиғалардың уақытпен бірге жазылған тарихы.

---

### 🔹 Қандай ақпарат жазылады?

Бұл жүйеде келесі ақпараттар жазылады:

* бағдарламаның іске қосылуы
* пайдаланушының жүйеге кіруі
* деректерді өзгерту әрекеттері
* қателер мен ескерту хабарламалары

---

### 🔹 Уақытты қалай қосамыз?

Уақыт қосу үшін Java тілінде:

* `LocalDateTime.now()` қолданылады
* уақытты әдемі көрсету үшін `DateTimeFormatter` пайдаланылады

---

## 💻 Қолданылған технологиялар

* Java
* FileWriter
* IOException
* LocalDateTime
* DateTimeFormatter
